## ✨ Features

### 👤 Biodata & Matching
- Create and manage your personal Islamic biodata profile
- Browse and discover potential matches based on compatibility
- Like, skip, or express interest in profiles — all within Islamic guidelines

### 💬 Chat & Communication
- Secure in-app messaging between matched users
- Supervised communication to maintain proper boundaries (*mahram*-aware design)
- Clear conversation etiquette aligned with Islamic values

### 🎓 Mawaddah Academy
- Pre-marriage learning center with curated educational content
- Courses and modules covering Islamic marriage principles, rights, and responsibilities
- Helps users prepare spiritually and practically before entering a marriage

### 🛡️ Role-Based Access
| Role | Capabilities |
|------|-------------|
| **User** | Register, create biodata, browse matches, chat, access Academy |
| **Admin** | Manage users, moderate content, oversee platform activity |

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18 or higher
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/taaruf-mawaddah.git

# 2. Navigate into the project directory
cd taaruf-mawaddah

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for Production

```bash
npm run build
```

The output will be in the `dist/` folder, ready to be deployed.

## 🙏 Islamic Context

> *"And of His signs is that He created for you from yourselves mates that you may find tranquility in them; and He placed between you affection and mercy."*
> — **Quran 30:21**

**Mawaddah** (مَوَدَّة) means *love* and *affection* in Arabic — one of the blessings Allah places between spouses. This platform is built with the intention of making the halal path of *taaruf* more accessible for Muslims in the digital age.
